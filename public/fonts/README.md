# Custom Fonts Setup

Please download and place the following font files in this directory:

1. **Milara** - Download from: https://www.dafont.com/milara.font
   - Place as: `Milara.ttf` or `Milara.otf`

2. **Apollo** - Download from: https://www.dafont.com/apollo.font
   - Place as: `Apollo.ttf` or `Apollo.otf`

3. **Driftune** - Download from: https://www.dafont.com/driftune.font
   - Place as: `Driftune.ttf` or `Driftune.otf`

4. **Moonrising** - Download from: https://www.dafont.com/moonrising.font
   - Place as: `Moonrising.ttf` or `Moonrising.otf`

## Font Usage in Sections:
- Milara: Hero Section
- Apollo: Projects Section
- Driftune: Process Section
- Moonrising: Testimonials Section
